02 / Module Reference

Every module,
decoded

🧠
Gemini Client
core/client.ts
Primary API wrapper for Google Generative AI. callBrain() uses gemini-2.5-flash (temp 0.1, JSON mode). Four-strategy JSON parse fallback: direct → strip fences → regex extract → success-key search.
callBrain()callEmbeddingAPI()parseModelResponse()
🔍
Hybrid Ranker
search/hybridRanker.ts
Scores every workspace file across 7 weighted signals. Detects query intent to boost file type preferences. Enforces token budget (80K tokens / 15 files). Embedding calls gated to save API quota.
hybridSearch()rerankByInstruction()detectQueryIntent()
🗺
Code Graph
graph/codeGraph.ts
Builds a workspace-wide AST dependency graph using the TypeScript Compiler API. Extracts imports, exports, functions, classes, interfaces, types. Builds reverse importedBy index for centrality scoring.
buildCodeGraph()findRelatedFiles()findSymbol()
🔢
Vector Store
search/vectorStore.ts
In-memory cache of text→float[] embeddings (768 dimensions). Prevents redundant Embedding API calls. cosineSimilarity() normalized from [-1,1] to [0,1].
getEmbedding()cosineSimilarity()
🔧
AST Patcher
patcher/astPatcher.ts
Applies line-range edits bottom-up (sorted desc by startLine) to prevent offset drift. Validates TypeScript files via getSyntacticDiagnostics() — only valid code is written to disk.
applyAstSafePatches()
🔄
Multi-Pass Patcher
patcher/multiPassPatcher.ts
Runs up to 3 validation passes checking import resolution, edit range integrity, cross-file dependencies. Critical errors trigger AI-driven correction via fixPatchErrors().
generateMultiPassPatches()validatePatches()
💬
Chat Panel
chat/panel.ts
VS Code WebviewViewProvider for the dark-themed chat sidebar. postLog() for status, postMessage() for AI responses. Uses native --vscode-* CSS variables for theme integration.
ChatViewProviderpostLog()postMessage()

Slash Commands
commands/slashCommands.ts
Dispatches /explain, /fix, /doc, /test, /commit. /explain and /doc call Brain directly. /fix returns an instruction object for the extension host to dispatch through the full pipeline.

Rules Loader
commands/rulesLoader.ts
Reads .om-ai-rules.md from workspace root and injects it as "Project Rules" into every Brain prompt — like .cursorrules. Supports scaffolding a default rules file.

Retry Loop
core/retryLoop.ts
Wraps callBrain() with up to 3 attempts. Each failure appends "Return valid JSON with changes array." before retrying. Returns first valid response or failure object.

Diff Preview
features/diffPreview.ts
GitHub-style side-by-side diff webview. Left = original, right = patched. Counts +additions/−deletions. Returns a Promise resolved only by user Apply or Cancel action.

Lightweight wrapper around vscode.workspace.findFiles returning all .ts/.tsx/.js/.jsx files, excluding node_modules, dist, .git. Feeds the hybrid ranker and code graph.

🧠
Gemini 2.5 Flash
Code Generation Brain
Primary model for generating code patches. Temperature 0.1 for determinism. JSON-only response mode. Four-strategy parse fallback handles malformed outputs.
🔢
text-embedding-004
Semantic Vectors
768-dimensional float vectors for semantic file similarity. Calls are gated behind keyword pre-filters (score > 0.02 or file < 20K chars) to conserve API quota.
🔷
TypeScript Compiler API
AST Analysis + Validation
Dual use: building the workspace code graph via ts.createSourceFile, and validating patched files via getSyntacticDiagnostics() before writing to disk.
🧩
VS Code Extension API
Runtime Environment
WebviewViewProvider for chat sidebar, window.createWebviewPanel for diff viewer, workspace.findFiles for discovery, workspace.fs.stat for recency scoring.
📦
@google/generative-ai
Official SDK
Handles authentication, request formatting, and model selection. Abstracts the HTTP layer for both Brain (generateContent) and Embedding (embedContent) calls.
⚡
Node.js 18+
Extension Host Runtime
fs module handles direct file I/O during patch application. path ensures cross-platform resolution. No bundler required for the extension host process.
